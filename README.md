# <img src='Workflow/icon.png' width='45' align='center' alt='icon'> 2FA Email Codes Alfred Workflow

Extract 2FA codes from Apple’s Mail.app with [Alfred](https://www.alfredapp.com/).

## Usage

Extract verification codes from Mail.app via the keyword `mailcode`.

![Extract verification code via keyword](Workflow/images/showcase_1.png)

* <kbd>↩</kbd> Copy code and paste to frontmost app.
